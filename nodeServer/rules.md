# 状态码

## 2xx

1. 200 返回成功
2. 201 新建或修改成功
3. 202 请求进入后台排队（异步任务）
4. 204 用户删除数据

## 3xx

- 一般是重定向问题

## 4xx

1. 400 请求有错误，服务器没有进行新建或修改数据的操作
2. 401 无权限
3. 403 禁止访问
4. 404 服务器没有进行操作
5. 406 请求格式错误
6. 410 请求资源被永久删除，且不会在得到
7. 422 创建对象，发生验证错误

## 5xx

1. 500 服务端内部错误

# 请求方式

- get
  - /collection 返回资源对象
  - /collection/resource 返回单个资源对象
- post 返回新生成的资源对象
- put 返回完整的资源对象
- patch 返回完整的资源对象
- delete 返回空文档
